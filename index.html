Cloud Backup

Connect Google Drive or OneDrive to back up your save files securely using your own account.

Google Drive
OneDrive
Setup: Go to console.cloud.google.com → New project → Enable Google Drive API → Credentials → Create OAuth 2.0 Client ID → Type: Desktop app.
Add http://localhost:9473/oauth/callback as an authorized redirect URI. Copy the Client ID below.
Not connected
Setup: Go to portal.azure.com → Azure Active Directory → App registrations → New registration.
In "Redirect URI" choose Mobile and desktop and enter http://localhost:9473/oauth/callback. Copy the Application (client) ID below.
